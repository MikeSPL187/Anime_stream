# Anime Stream

Single-user anime streaming app built toward Crunchyroll-like mobile product maturity.

Current scope:
- Home as re-entry and discovery
- Search for explicit intent discovery
- Browse and Catalog for deeper exploration
- Series as an operational watch hub
- Player with progress continuity, episode handoff, quality control, and offline playback
- Watchlist, Downloads, History, and playback preferences for one user
